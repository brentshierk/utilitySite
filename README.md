## Building

1. set up base files

    ```
    git clone https://github.com/brentshierk/utilitySite
    cd hugo-app
    git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke

    ```



1. Run the application!
    - `hugo server -D`
